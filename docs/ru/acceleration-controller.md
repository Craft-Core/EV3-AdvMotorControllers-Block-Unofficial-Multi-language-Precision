# OFDL Acceleration Controller — Руководство по использованию

Генерирует плавный профиль скорости: **разгон** от `MinPwr` до `MaxPwr` за `AccelDist` градусов, движение на `MaxPwr` в середине, затем **торможение** обратно до `MinPwr` за `DecelDist` градусов — всё на основе показаний энкодера в реальном времени.

---

## Концепция

```
Encoder position →

0 ────── AccelDist ──────── (FullDist − DecelDist) ──────── FullDist
│ ramp up │       full speed        │      ramp down      │
MinPwr → MaxPwr              MaxPwr              MaxPwr → MinPwr
```

Блок **выводит значение скорости** — подключите его к входу мощности мотора на каждой итерации цикла. Он не управляет моторами напрямую.

---

## Настройка

### Шаг 1 — Блок конфигурации (запускается один раз перед циклом)

| Параметр | Описание | Типичное значение |
|----------|----------|-------------------|
| **EncPort** | Порты моторов с энкодером | `1.B+C` |
| **MaxPwr** | Максимальная скорость (−100 до 100) | `90` |
| **MinPwr** | Минимальная скорость в начале/конце (−100 до 100) | `15` |
| **AccelDist** | Дистанция разгона (в градусах) | `300` |
| **DecelDist** | Дистанция торможения (в градусах) | `300` |
| **FullDist** | Общая дистанция движения (в градусах) | `1000` |

### Шаг 2 — Блок вывода (запускается на каждой итерации цикла)

#### Выходные данные

| Выход | Описание |
|-------|----------|
| **Output** | Текущая скорость для подачи на моторы |
| **E1Output** | Текущее показание энкодера 1 (в градусах) |
| **E2Output** | Текущее показание энкодера 2 (в градусах) |
| **Finish** | `True`, когда достигнута дистанция `FullDist` |

---

## Режимы

| Режим | Описание |
|-------|----------|
| `Configuration` | Задать все параметры (вызвать один раз перед циклом) |
| `ACDC_Output1Degs` | Вывод скорости только по энкодеру 1 |
| `ACDC_Output2Degs` | Вывод скорости по обоим энкодерам (усреднение) |

---

## Типичная структура цикла

```
[Reset encoders (Advanced Encoder block)]
[Configuration: EncPort=B+C, MaxPwr=90, MinPwr=15, AccelDist=300, DecelDist=300, FullDist=1000]

Loop until Finish = True:
  [ACDC_Output2Degs] → Output, E1Output, E2Output, Finish
  [Drive motors with Output]
  [PD Controller if line following]
```

---

## Советы

- Всегда **сбрасывайте энкодеры** перед началом (используйте блок `EncReset` из Advanced Encoder).
- `AccelDist + DecelDist` должно быть меньше `FullDist`, иначе профили перекрываются.
- Комбинируйте с **PD-контроллером** (режимы `PDpwr_*`) для плавного слежения за линией с переменной скоростью: подайте `Output` как вход мощности в блок PD.
- Используйте `ACDC_Output1Degs` для задач с одним мотором; используйте `ACDC_Output2Degs` для двухмоторных приводов, чтобы компенсировать дрейф энкодеров.
